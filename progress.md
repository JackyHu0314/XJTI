## 版本

### v0.0.2 — 2026-04-18

**交互彩蛋与陕西美食推荐**

新增功能：
- **食谱弹窗**：点击结果页顶部人格代码 → 弹出专属陕西美食推荐（10 人格 × 4 道菜），采用动态卡片风格（hover 时 emoji 放大浮动）
- **6 个交互彩蛋**：
  1. 双击雷达图 → 雷达拉满动画（0.6s 过渡）+ 10 个人格代码从中心散开
  2. 点击"闪闪发光" → 金色星雨（80 颗，4.5-8s 渐隐）
  3. 点击"色彩" → 七彩星雨（90 颗，赤橙黄绿青蓝紫）
  4. 点击"慢行者" → 梧桐叶飘落（12 片，随机 1 片带"⚠ 小心头顶"贴纸）
  5. 点击"长安" → 乌龟慢爬（8.5s，带"真的很远……"气泡）
  6. MEAL 限定：点击人格代码额外触发食堂菜 emoji 雨（60 个）

文本优化：
- 封面欢迎语去除所有中文标点并换行
- 结果页 OUTRO 文字分段空行 + 关键词加粗（交大、极客、慢行者、人格、学府、色彩、长安、自己、闪闪发光）
- 人格描述首行缩进 2 字符
- 页面底部新增"本页含多个动效彩蛋，同学们自行探索"提示（居中，footer 上方）

技术细节：
- 新增 `RecipeModal.tsx` 组件 + `recipes.ts` 数据源（10 人格 × 4 道陕西美食）
- `RadarChart` 支持 `maxed` 状态（双击时 0.6s 缓动过渡到满级）
- 所有彩蛋浮层用 `createPortal` 挂到 `document.body`，避免 `.result` 的 `transform` 祖先定位问题
- 星雨亮度曲线优化（35%-70% 最亮，适配手机短屏）
- 所有动画结束自动卸载 DOM，不污染海报截图
- README 新增委婉的 star 提示（"为学生会活动准备的小工具，如果你觉得有趣或有用，欢迎 star 支持一下"）

部署：
- 构建产物：`dist/` 约 368KB（gzip 后 108KB）
- 托管平台：Cloudflare Pages，绑定域名 xjti.top

**说明**：v0.0.2 是交互增强版本，在 v0.1.0 基础上新增 6 个彩蛋 + 陕西美食推荐，提升趣味性和地方特色。

## 决策记录

- **React + TS 而非原生 JS**：用户明确要求 React + TypeScript + Vite 架构
- **不引 React Router**：三视图用 `useState` 切换，无需路由（学生会场景分享的是海报图片，不是链接）
- **不引 Tailwind / chart.js**：雷达图纯 SVG 手绘，样式用原生 CSS，保持极简
- **保留 html2canvas**：海报截图是学生会传播的核心（小红书 / 朋友圈），这是唯一运行时依赖
- **13 题是硬约束**：用户明确要求。从 70 题候选里精选，每个人格出现在 2-3 题里
- **数据单一源**：`src/data/questions.ts` + `src/data/personalities.ts` + `src/data/recipes.ts`（v0.0.2 新增）是题库 / 人格 / 食谱的唯一真相来源
- **敏感话题黑名单**：学生会场景下避免争议，从候选题中筛选 / 改写时必须过这道关
- **130 年 / 西迁 70 年彩蛋**：13 题 / 70 切片 / 1896 → 2026，数字暗合，不喊口号，懂的人自然懂
- **文本内容/ 不上传**：该目录含用户原始素材（未公开），`.gitignore` 必须拦住
- **彩蛋用 createPortal**（v0.0.2）：所有浮层挂到 `document.body`，避免 `.result` 的 `transform` 祖先定位问题
- **陕西美食推荐**（v0.0.2）：每个人格 4 道菜，结合人格特点 + 陕西地方特色（肉夹馍、biangbiang面、冰峰、羊肉泡馍、胡辣汤、茯茶等）
